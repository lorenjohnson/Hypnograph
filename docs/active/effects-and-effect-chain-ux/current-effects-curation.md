# Current Effects Curation

This is the working inventory for the base effects currently available from the `Add Effect` menu. These are the underlying building blocks, not the packaged effect chains.

This set also needs curation and refinement before release. The goal is a standard built-in list that is versatile, wide-ranging, and worth learning. Eventually this area should also gain documentation for what each effect does, when to use it, and what other effects it pairs well with.
