# Docs

This repository contains two macOS apps:

- `Hypnograph/` (target: `HypnographApp`)
- `Divine/` (target: `DivineApp`)

Both share `HypnoCore/` (and `HypnoUI/`).

## Layout

- `docs/Hypnograph/architecture/` — architecture notes
- `docs/Hypnograph/projects/` — planning docs and historical implementation notes
- `docs/Hypnograph/reference/` — user-facing reference (e.g., controls)
- `docs/ontology/` — shared core ontology + diagrams (generated)
- `docs/Divine/` — Divine app docs
