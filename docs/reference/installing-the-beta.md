# Installing The Hypnograph Beta

Because Hypnograph is currently distributed as a direct beta download during early development, macOS shows an extra security warning the first time you open it. This is part of Apple's normal protection for apps that are not yet distributed through Apple's standard beta or notarized channels.

## Steps

1. Download the current [Hypnograph beta](https://github.com/lorenjohnson/Hypnograph/releases/download/v0.2.0-beta.11/Hypnograph-0.2.0-11-macOS-unsigned.dmg). You may already have done this from the homepage.
2. Open the downloaded file, drag the Hypnograph icon into the `Applications` folder, and then open Hypnograph from `Applications`.
4. If macOS shows a warning with only `Move to Trash` or `Cancel`, choose `Cancel`.
5. Open `System Settings` > `Privacy & Security`.
6. Scroll down until you see the message about Hypnograph, then click `Open Anyway`.
7. Open Hypnograph again. macOS should show one more warning, this time with an `Open Anyway` option.
8. Click `Open Anyway`, then enter your password if macOS asks for it.
9. After that first approval flow, Hypnograph should open normally on subsequent launches, until you install a newer beta release and need to approve it once again.

You should only need to do this the first time you open a new beta release.

If you want Apple's own instructions as well, see [Open an app by overriding security settings](https://support.apple.com/en-afri/guide/mac-help/mh40617).

This extra setup is temporary. Hypnograph is expected to move to Apple's beta distribution platform soon, which should remove these extra steps. Thanks for your patience as an early tester. The upside, for now, is getting access to the app sooner.
