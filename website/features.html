Feature Catalog

A broader look at what Hypnograph can do right now.

This list covers current capabilities in the beta build. Some interactions are still evolving, but the core workflow is already usable for exploratory visual practice.

Apple Photos Integrated

When Apple Photos access is granted, Hypnograph works directly with your library and saves Hypnograms snapshots, favorites, and exclusions back into Apple Photos under a dedicated Hypnographs folder.

File-Based Source Ingest

Work with plain file-based media sources without forcing a new library format.

Montage + Sequence Modes

Switch between generation styles to emphasize drifting remix behavior or more directed clip sequencing.

Transition Styles + Timing

Use multiple transition options and duration controls to make clip changes feel fluid, immersive, and intentionally hypnotic.

People-Aware Recomposition

Compositions can reframe based on where people are in the image, helping keep human subjects central while sequences evolve.

Intelligent Fullscreen Reframing

Fill-screen output intelligently reframes media to preserve as much image content as possible, including making vertical footage feel intentional inside horizontal compositions.

Layered Composition

Stack multiple source and generated elements, then reorder and blend them into a single output stream.

Blend Mode Routing

Cycle and tune blend relationships between layers to push depth, contrast, and atmosphere.

Metal-Native Effect Chains

Effects are Apple Metal-native shaders with extensive parameter control, designed to be chained for entirely custom looks.

Global + Per-Layer Control

Work both locally on a single layer and globally across the output, with numerous tweakable parameters for precise visual shaping.

Solo + Isolate Workflows

Temporarily isolate layers while refining effects, then reintroduce full composition context.

Live Preview + Display Output

Use built-in live display controls to decouple preview from output, so you can keep composing privately and send selected results to a dedicated live screen.

Clean Screen + Workspace Toggle

Press Tab for a clean viewing mode, then return to a rich tool workspace for effects, format, and composition controls.

Playback Steering

Move through sources, trigger new clip choices, and redirect flow quickly during active sessions.

Guided Randomization + Manual Picking

Set practical generation boundaries, then manually choose Apple Photos selections or file sources whenever you want precise direction.

Saveable Effect Sets

Save and reopen effect chains as reusable sets, so complex, parameter-rich looks can be recalled, refined, and evolved over time.

Hypnogram Reference Saves

Save compositions as lightweight hypnogram references and reopen them later for replay and re-rendering without keeping duplicate full video files.

Render Resolution Options

Choose render resolution by output need, including 720p, 1080p, and 4K, so exports can be optimized for smaller files or higher-fidelity delivery.

Favorites + Exclusions

Mark material to preserve or avoid so randomization becomes more intentional over repeated use.

Keyboard + Controller Support

Operate core playback and composition controls with shortcuts and game controller mappings.
