## 🎨 Control Philosophy

**BUMPERS = EFFECTS**
- LB (Left Bumper): Cycle **Blend Mode** (current layer)
- RB (Right Bumper): Cycle **Source Effect** (current layer)

**TRIGGERS = STYLE/RESET**
- LT (Left Trigger): Clear all effects & reset blend modes
- RT (Right Trigger): Toggle **Montage/Sequence** style (Dream mode)

**FACE BUTTONS = ACTIONS**
- X (left): Cycle **Global Effect** (entire composition)
- Y (top): Snapshot (quick PNG capture)
- A (bottom): New composition
- B (right): Save/Render video

## 🎯 Common Workflows

### Creating a New Composition
1. Press **A** to generate new random composition
2. Use **D-Pad ←/→** to navigate sources
3. Press **LB** to cycle blend modes on current layer
4. Press **RB** to cycle effects on current layer
5. Press **X** to cycle global effects
6. Press **B** to save/render when satisfied

### Quick Adjustments
- **LB** (Left Bumper) - Quickly cycle through blend modes on current layer
- **RB** (Right Bumper) - Quickly cycle through effects on current layer
- **X** button - Quickly cycle through global effects

### Snapshot vs Save
- **Y** button - Quick snapshot (PNG, instant)
- **B** button - Full render (video, takes time)

### Navigation
- **D-Pad ←/→** - Navigate between sources
- **1-9** on keyboard - Jump directly to source 1-9

## 💡 Tips for Gallery Installation

1. **Keep HUD visible** - Press **Back/Options** to toggle
2. **Watch mode** - Press **L3** to enable auto-generation (screensaver mode)
3. **Pause to examine** - Press **Start/Menu** to pause/play
4. **Quick snapshots** - Press **Y** to capture interesting moments
5. **Instrument-like play** - Use bumpers for real-time effect manipulation

## 🔧 Setup

1. **Pair controller** via Bluetooth:
   - System Settings → Bluetooth
   - Hold Xbox button + Sync button on controller
   - Click "Connect" when it appears

2. **Launch Hypnograph** - Controller auto-detected

3. **Start playing!** - All buttons work immediately

Console will show: `🎮 Game controller connected: Xbox Wireless Controller`
